# Malaysia Restaurant Ordering System

This workspace contains separate projects as requested:

- `restaurant-order-system` - Java 17 Spring Boot backend.
- `restaurant_waiter_app` - Flutter waiter mobile APP.
- `restaurant_cashier_pc` - Vue3 cashier PC client.
- `restaurant_admin_pc` - Vue3 admin portal.

The three client systems are independent projects and do not share one frontend project.
