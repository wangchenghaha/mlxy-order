# restaurant_cashier_pc

Independent Vue3 + Vite + Element Plus cashier PC project.

Main workflows:

- Cashier login.
- Query pending checkout, served, and paid orders.
- Unified checkout with five Malaysia payment methods: cash, bank card, DuitNow QR, Touch 'n Go, Alipay cross-border.
- Receipt preview, reprint submission, dynamic backend i18n.

```bash
npm install
npm run dev
```
